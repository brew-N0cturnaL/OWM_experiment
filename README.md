# simple_weather
A program that allows to get information about weather and ASCII-art of weather conditions by entering a city name using OpenWeatherMap API.
To get started, download all .txt and .py files and place them in one folder, otherwise you'd have to write the whole way to them ASCII-arts. 
Execute the program, enter a city, get a dope weather conditions art and some detailed information. As simple as plug-n-play.
